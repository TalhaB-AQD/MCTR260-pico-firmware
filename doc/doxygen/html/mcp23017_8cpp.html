Detailed Description

MCP23017 16-bit I2C GPIO Expander: driver implementation.

Controls the two MCP23017 chips on the MechaPico MCB:

  • U6_1 (0x20): Stepper motors M1-M5, enable, microstepping, SPREAD/PDN
  • U6_2 (0x21): DC motors 3-4, LEDs, general 5V I/O

I2C PERFORMANCE (400kHz bus, 3-byte writes): Single register write: ~50uss (start + addr + reg + data + stop) stepperPulse() per motor: ~100uss (set + clear = 2 writes) stepperPulseBatchPortB(): ~100uss total for ALL 4 motors (2 writes) -> Batching saves ~300uss per Core 1 update cycle

See mcp23017.h for the full MechaPico MCB pin mapping diagrams.
