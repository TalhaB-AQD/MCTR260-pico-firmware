Detailed Description

Stepper motor configuration (one per physical motor).

Each motor gets its own config struct with driver type, mechanical parameters, and motion limits. These are defined in project_config.h and passed to the MotorStepper constructor during motors_init().

Note
On the MechaPico MCB, all stepper pins are on the MCP23017 I2C expander, NOT on Pico GPIO. The legacy pinStep/pinDir/pinEnable fields are set to -1 and ignored.
See also
project_config.h where these values are set for your robot
mcp23017.h for the physical pin mapping on the MCP23017

◆ acceleration

float MotorStepperConfig::acceleration

Acceleration rate in steps per second squared. Controls how quickly the motor ramps up to target speed. Too high = missed steps (motor stalls). Too low = sluggish response. Default: 8000.

◆ direction

int8_t MotorStepperConfig::direction

Direction multiplier: 1 for normal, -1 to reverse. Like DC motors, use this to correct for motors mounted in opposite orientations.

◆ index

uint8_t MotorStepperConfig::index

Motor index (0-4 for M1-M5 on MCP23017). For a 4-motor mecanum: 0=FL, 1=FR, 2=BL, 3=BR. Motor 4 is a spare.

◆ maxSpeed

float MotorStepperConfig::maxSpeed

Maximum speed in steps per second. The motor will never exceed this rate, even if the joystick is pushed to maximum. Higher values need faster I2C communication. Default: 4000.

◆ microstepping

uint8_t MotorStepperConfig::microstepping

Microstep divisor (1, 2, 4, 8, 16, 32). Higher values = smoother motion but lower torque and higher step rates needed for the same RPM. Example: 8 microsteps with a 200-step motor gives 1600 effective steps per revolution.

◆ pinDir

int8_t MotorStepperConfig::pinDir

Legacy: Direction pin. Set to -1 on MechaPico MCB. Direction is set via MCP23017 Port B bits.

◆ pinEnable

int8_t MotorStepperConfig::pinEnable

Legacy: Enable pin. Set to -1 on MechaPico MCB. All steppers share one enable signal on MCP23017 GPA3 (STPR_ALL_EN).

◆ pinStep

int8_t MotorStepperConfig::pinStep

Legacy: Step pulse pin. Set to -1 on MechaPico MCB because step pulses go through MCP23017, not Pico GPIO.

◆ stepsPerRev

uint16_t MotorStepperConfig::stepsPerRev

Full steps per revolution. Standard stepper motors have 200 steps/rev (1.8 degrees per step). Some high-resolution motors use 400.
