Detailed Description

Wheel speeds for a 4-wheel mecanum drive.

Contains the computed speed for each wheel after inverse kinematics. Values are normalized to the range [-1.0, +1.0], where 1.0 represents maximum motor speed in that direction.

Note
The wheel naming follows the standard convention when looking DOWN at the robot from above: FL (front-left) | FR (front-right) BL (back-left) | BR (back-right)
